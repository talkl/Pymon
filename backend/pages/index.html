Welcome to PYmon

PYmon is an online multiplayer version of the famous Hasbro game simon
The rules are simple...
  1. Click the start button below
  2. Select a user name
  3. Join or create a new game
  4. Wait for other players to join
  5. Once all players listened to the sequence the game starts
  6. On your turn, press the correct light button according to the sequence
  7. If you failed, you'r out...
  8. Once the sequence completed all the remaining players are winners! Yay!
  9. Try to win as many games as possible
